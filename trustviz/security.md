# Security Policy

## Purpose
TrustViz is an **educational** system. It verifies math/visualizations and blocks dual-use or malicious content.

## Out-of-scope / Disallowed
- Malware, exploit development, payload crafting, privilege escalation, botnets, keyloggers, shellcode, CVE weaponization.
- Arbitrary code execution outside the sandbox.

## Reporting a Vulnerability
If you discover a policy bypass, unsafe output, or sandbox escape:
1. Do **not** open a public issue with exploit details.
2. Email: SofiaTejada001@gamil.com with steps to reproduce.
3. We aim to acknowledge within 7 days and fix promptly.

## Disclosure
Please allow a reasonable time for remediation before public disclosure.
